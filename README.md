## 📋 Sobre o Desafio

Boas vindas ao primeiro desafio da Alura!  
Este projeto tem como obetivo ajudar o **Senhor João** a decidir qual loja da sua rede **Alura Store** vender para iniciar um novo empreendimento. Para isso, analisamos dados de vendas, desempenho e avaliações das 4 lojas fictícias da Alura Store. O objetivo é identificar a loja com menor eficiência e apresentar uma recomendação final baseada nos dados.

## 🎯 Objetivo

Identificar a loja com menor eficiência operacional e comercial através da análise de dados reais de vendas, apresentando uma recomendação final baseada em evidências.

## 🛠️ Tecnologias Utilizadas

- **Python 3.11** - Linguagem principal
- **Pandas** - Manipulação e análise de dados
- **Matplotlib** - Visualização de dados e criação de gráficos
- **Jupyter Notebook** - Ambiente de desenvolvimento interativo


## 📊 Análises Realizadas

### 1. **Métricas de Performance por Loja**
- Faturamento total e ticket médio
- Avaliações dos clientes
- Frete médio por venda
- Quantidade de vendas

### 2. **Análise de Produtos**
- Top 5 produtos mais vendidos
- Top 5 produtos menos vendidos
- Produtos por categoria

### 3. **Análise por Categoria**
- Faturamento por categoria
- Quantidade de vendas por categoria
- Avaliação média por categoria

### 4. **Análise de Frete**
- Frete médio por loja
- Relação preço vs frete
- Custo de frete por categoria
